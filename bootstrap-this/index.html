What is Bootstrap?

Bootstrap is a free and open-source front-end framework that provides ready-made components to be used in your project. With it, you can create and customize responsive websites for mobile devices and desktops, with pre-built components and powerful plugins.

Basically, the framework structure is formed by two directories:

  • css: which contains the files needed to style the elements and an alternative to the original theme;
  • js: contains versions of the bootstrap.js file (original and minified), which is responsible for running style applications that need interactive manipulation.

To assign a characteristic to an element, just inform the corresponding class in the element's class property to be styled. See the example: 

<img src="image-name.jpg" alt="alternative text" class="rounded-sm" />

In this example, the content rounded-sm has been assigned to the image property. This is a style that adds rounded corners to the element. Therefore, when loading the image, the characteristics referring to that class will be applied to the element.

Carousel

A commonly used component in Bootstrap is Carousel. It is a slideshow, that is, a tool that allows the display of images in a responsive way. It also makes it possible to include special effects for image transition and display controls such as next and previous indicators.

Navbar

Another powerful framework component is Navbar, which allows you to build a responsive navigation system. It is possible to configure different ways of displaying the menu, such as choosing between lateral or top positioning and defining the display form, which can be extended or contracted.

It is also possible to determine how the menu links are displayed, which can be in the form of a button, link, drop-down menu, among other settings, to facilitate the implementation of site navigation.
